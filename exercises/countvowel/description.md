# Exercice : Count Vowel

## Objectif
Compter le nombre de voyelles dans une chaîne.

## Consignes
- Lire une chaîne en entrée.
- Compter les voyelles : a e i o u y (minuscules uniquement pour cet exercice).
- Afficher le nombre.

## Exemple
Input :
hello

Output :
2

## Contraintes
- Le fichier doit s'appeler **countvowel.py**
