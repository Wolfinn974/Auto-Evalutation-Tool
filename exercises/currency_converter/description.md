# Exercice : Currency Converter

## Objectif
Convertir un montant en euros vers des dollars américains.

## Consignes
- Lire un montant en euros (float).
- Le taux est fixé à : 1€ = 1.07$
- Afficher le montant converti.

## Exemple
Input :
10

Output :
10.7

## Contraintes
- Le fichier doit s'appeler **currency_converter.py**
