# Exercice : E-commerce PayPal Commission Calculator

## Objectif
Calculer le montant réel reçu après la commission PayPal.

## Consignes
- Lire un montant en euros.
- Commission : 2.9% + 0.30€
- Afficher le montant final après commission.

## Exemple
Input :
100

Output :
96.8

(calcul : 100 - (100 * 0.029 + 0.30))

## Contraintes
- Le fichier doit s'appeler **ecommerce_paypal_commission.py**
