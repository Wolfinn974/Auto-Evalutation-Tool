# Exercice : FizzBuzz

## Objectif
Créer un programme qui génère une séquence avec des remplacements conditionnels.

## Consignes
1. Lire un entier positif `n`.
2. Pour chaque nombre de 1 à `n` :
   - Afficher "FizzBuzz" si le nombre est divisible par 15.
   - Afficher "Fizz" si le nombre est divisible par 3.
   - Afficher "Buzz" si le nombre est divisible par 5.
   - Sinon afficher le nombre.
3. Chaque élément doit être affiché sur une nouvelle ligne.

## Exemple
Input :
5

Output :
1
2
Fizz
4
Buzz

## Contraintes
- Le fichier soumis doit s'appeler **fizzbuzz.py**
- Pas d'espace ou texte supplémentaire.
- Pas d’interaction utilisateur autre que l’input demandé.

## Conseils du prof (gentils)
Fais simple. Une boucle, quelques conditions, et on respire.
