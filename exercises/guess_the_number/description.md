# Exercice : Guess the Number

## Objectif
Comparer un nombre saisi avec un nombre cible.

## Consignes
- Le nombre cible est 42.
- Lire un entier.
- Afficher :
  - "too low" si le nombre est inférieur à 42.
  - "too high" si le nombre est supérieur à 42.
  - "correct" si le nombre est 42.

## Exemple
Input :
42

Output :
correct

## Contraintes
- Le fichier doit s'appeler **guess_the_number.py**
