# Exercice : Mini Calculatrice

## Objectif
Créer une petite calculatrice prenant deux nombres et un opérateur.

## Consignes
- Lire un nombre, un opérateur, puis un deuxième nombre.
- Opérateurs autorisés : + - * /
- Afficher le résultat.
- Division entière avec décimales (ex : 5 / 2 → 2.5)

## Exemple
Input :
5
*
3

Output :
15

## Contraintes
- Le fichier doit s'appeler **minicalculatrice.py**
