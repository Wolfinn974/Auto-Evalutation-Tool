# Exercice : Mystery Word

## Objectif
Compter combien de lettres correspondent à un mot secret.

## Consignes
- Le mot secret est : "python"
- Lire un mot en entrée.
- Afficher le nombre de lettres qui correspondent **au même index**.

## Exemple
Input :
pytxon

Output :
4

(positions correctes : p y t _ o _)

## Contraintes
- Le fichier doit s'appeler **mystery_word.py**
