# Exercice : Print Alphabet

## Objectif
Afficher l'alphabet en lettres minuscules dans l'ordre croissant.

## Consignes
- Ne rien lire en entrée.
- Afficher : abcdefghijklmnopqrstuvwxyz
- Sans espaces, sans sauts de ligne supplémentaires.
- Output exact.

## Exemple
Output :
abcdefghijklmnopqrstuvwxyz

## Contraintes
- Le fichier doit s'appeler **printalphabet.py**
- Pas de print avec texte additionnel.
- Pas d'espace.
- Un seul print ou équivalent.

## Conseils
range(97, 123) ou direct une string, fais ta vie.
