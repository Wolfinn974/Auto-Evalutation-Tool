# Exercice : Print Reverse Alphabet

## Objectif
Afficher l'alphabet en lettres minuscules, dans l'ordre décroissant.

## Consignes
- Ne rien lire en entrée.
- Afficher : zyxwvutsrqponmlkjihgfedcba
- Sans espaces, sans texte additionnel.
- Un seul saut de ligne final.

## Exemple
Output :
zyxwvutsrqponmlkjihgfedcba

## Contraintes
- Le fichier doit s'appeler **printreversealphabet.py**
